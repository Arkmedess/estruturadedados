# Estudos de Estrutura de Dados

Este diretório contém um guia de estudo prático baseado no material existente dos notebooks de Estrutura de Dados.

Arquivos:
- `guia_de_estudo.md`: visão geral dos principais temas, conceitos e como usar o material para prova.
- `cheatsheet.md`: referência rápida de sintaxe, métodos e comparações de estruturas de dados.
- `cola_prova.ipynb`: cola completa em estilo Jupyter, com teoria didática e exemplos de código por aula.
- `INSTALACAO_VENV.md`: passo a passo rápido para criar e usar o ambiente virtual (venv) com Jupyter.
